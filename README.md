# Netflix Content Strategy Analysis (Python)

## 📌 Project Overview

This project analyzes Netflix content performance and strategy using Python. The goal is to uncover insights about what types of content perform best based on factors such as language, content type, release timing, and global availability.

By leveraging data analysis techniques, this project aims to provide a deeper understanding of Netflix's content strategy and audience preferences.

---

## 📊 Dataset Description

The dataset used in this project contains the following features:

* **Title** – Name of the content
* **Release Date** – Date when the content was released
* **Hours Viewed** – Total number of hours the content has been watched
* **Language** – Original language of the content
* **Content Type** – Movie or TV Show
* **Available Globally** – Whether the content is available worldwide

---

## 🎯 Objectives

* Analyze which types of content generate the most viewership
* Identify trends based on release dates
* Compare performance across different languages
* Evaluate the impact of global availability on viewership
* Understand differences between movies and TV shows

## 🔍 Key Analysis Performed

### 1. Data Cleaning

* Handling missing values
* Converting date columns to datetime format
* Standardizing categorical variables

### 2. Exploratory Data Analysis (EDA)

* Distribution of hours viewed
* Content type comparison (Movies vs TV Shows)
* Language-based performance analysis
* Global vs non-global availability trends

### 3. Trend Analysis

* Year-wise content release trends
* Seasonal performance patterns
* Total Viewership trend by content type
* Weekly and Monthly release patterns and viewership

### 4. Visualization

* Bar charts for content comparison
* Line graphs for trend analysis
* Combo charts for comparison

---


## 📈 Insights & Findings

* Certain languages dominate in terms of total watch hours
* Globally available content tends to perform better
* TV shows often generate higher cumulative watch time compared to movies
* Release timing plays a key role in content performance
